## ATS vs ETS2 Differences

**Prompt:**
> What is different about modding for ATS compared to ETS2? Are the unit file formats the same?

**What the skill reads:**
- `docs/engine/units.md`
- `docs/engine/formats.md`
- Source: https://modding.scssoft.com/wiki/Documentation/Engine/Units

**Note:** ATS and ETS2 share the same engine and unit format system. Most modding documentation applies to both games. ATS-specific differences are noted where they exist.

## ATS Configuration Variables

**Prompt:**
> What configuration variables are available for ATS? How do I use g_pounds?

**What the skill reads:**
- `docs/engine/config_variables/g_pounds.md`
- `docs/engine/config_variables/g_currency.md`
- `docs/engine/configuration_variables.md`

**What you get:** `g_pounds` switches between metric and imperial weight units — relevant for ATS which defaults to imperial. `g_currency` sets the in-game currency type.

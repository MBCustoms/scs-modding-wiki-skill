# ETS2 Truck Modding — Example Prompts

These examples demonstrate how to use the scs-modding-wiki skill for Euro Truck Simulator 2 truck modding.

---

## Engine Accessory

**Prompt:**
> How do I define a truck engine in a .sii file? What parameters are required?

**What the skill reads:**
- `docs/engine/units/accessory_engine_data.md`
- Source: https://modding.scssoft.com/wiki/Documentation/Engine/Units/accessory_engine_data

**What you get:** Complete `accessory_engine_data` unit definition with all required and optional fields — torque, power, RPM curves, fuel consumption, and more — with exact `.sii` syntax.

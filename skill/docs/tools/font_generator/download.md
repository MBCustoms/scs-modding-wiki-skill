# Font Generator Download

Provides support for generating game font file. See the included example file.

- For game versions 1.43 and above use [this](https://download.eurotrucksimulator2.com/fontgen_211223.zip).
- For older game versions use [this](https://download.eurotrucksimulator2.com/fontgen_190425.zip).

1.43+ font rendering system documentation is described [here](/wiki/Documentation/Engine/Fonts "Documentation/Engine/Fonts").

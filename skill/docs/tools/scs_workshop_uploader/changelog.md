# SCS Workshop Uploader Changelog

**Source:** https://modding.scssoft.com/wiki/Documentation/Tools/SCS_Workshop_Uploader/Changelog

## Overview

This page documents the complete changelog for the SCS Workshop Uploader tool. The tool supports multiple versions:

- **Current Version**: Compatible with the latest released game version
- **Public Beta Version**: Compatible with open/experimental beta versions
- **Historical Versions**: Available through Steam beta branches for older game versions
- **Legacy Versions**: Very old versions consolidated in a single branch

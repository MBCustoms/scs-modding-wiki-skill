# Steam formatting

## Syntax and Preview

| Syntax | Preview |
|--------|---------|
| `[h1]Header text[/h1]` | Header text |
| `[b]Bold text[/b]` | **Bold text** |
| `[u]Underlined text[/u]` | Underlined text |
| `[i]Italic text[/i]` | _Italic text_ |
| `[strike]Strikethrough text[/strike]` | ~~Strikethrough text~~ |
| `[noparse]Doesn't parse [b]tags[/b][/noparse]` | Doesn't parse [b]tags[/b] |
| `[url=scssoft.com]Website link[/url]` | [Website link](http://scssoft.com) |
| `[list][*]Bulleted list[*]Bulleted list[*]Bulleted list[/list]` | • Bulleted list<br>• Bulleted list<br>• Bulleted list |
| `[olist][*]Ordered list[*]Ordered list[*]Ordered list[/olist]` | 1. Ordered list<br>2. Ordered list<br>3. Ordered list |
| `[quote=author]Quoted text[/quote]` | > Originally posted by **author**:<br>> Quoted text |
| `[code]Fixed-width font, preserves spaces[/code]` | Fixed-width font, preserves    spaces |
| `[img]image_url[/img]` | ![Image](image_url) |

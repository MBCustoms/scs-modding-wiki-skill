# Documentation/Tools/SCS Workshop Uploader/Validation

Each mod must pass validation before entering the workshop. This process prevents mods that could cause player issues due to preventable problems.

When validation completes with warnings or errors, a log window displays for inspection. The validation log is also written to a file at these default locations:

## Windows

`C:/Users/<your_user_name>/My Documents/SCS Workshop Uploader/validation.log.txt`

**Note:** With Windows 10 and OneDrive, the path becomes `C:/Users/<your_user_name>/OneDrive/Documents/SCS Workshop Uploader/validation.log.txt`

## Linux

`~/.local/share/SCS Workshop Uploader/validation.log.txt`

## Mac OS X

`/Users/<your_user_name>/Library/Application Support/SCS Workshop Uploader/validation.log.txt`

Replace `<your_user_name>` with your actual user name.

For detailed information about specific validation issues, refer to the "Validation warning and error codes" page, which provides codes to help identify and resolve validation failures.

# Documentation/Tools/SCS Workshop Uploader/tool.log.txt

The engine log for the SCS Workshop Uploader contains more information about errors and warnings.

## Location

### Windows

```
C:/Users/<your_user_name>/Documents/SCS Workshop Uploader/tool.log.txt
```

**Note:** In case of Windows 10 and OneDrive usage the path will be:
```
C:/Users/<your_user_name>/OneDrive/Documents/SCS Workshop Uploader/tool.log.txt
```

### Linux

```
~/.local/share/SCS Workshop Uploader/tool.log.txt
```

### Mac OS X

```
/Users/<your_user_name>/Library/Application Support/SCS Workshop Uploader/tool.log.txt
```

# Documentation/Tools/SCS Workshop Uploader/FAQ

## Frequently Asked Questions

**Q: I want to upload my map mod to the Workshop however I get many errors that .aux and other file formats are not supported. Help!**

A: In the initial release of the Steam Workshop implementation, map mods are not supported. Future support may be added.

**Q: I want to create a mod of type X however I don't see this type tag available. What to do?**

A: The "Others" tag is recommended as a temporary solution. Additional tags will be added over time.

**Q: The game was updated and my mod is not working anymore. How can I provide multiple versions of my mod to users on different versions?**

A: Steam Workshop mods support multiple game versions simultaneously. Refer to the Workshop mod structure documentation for implementation details.

**Q: I want to update my mod. How can I do this?**

A: Open the SCS Workshop Uploader and select a previously published mod. Choose your mod folder, select an image, and write a changelog entry. Click "Upload" to automatically distribute the update to all subscribers. See the tutorial on updating workshop mods for additional guidance.

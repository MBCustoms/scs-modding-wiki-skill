# Documentation/Tools/SCS Blender Tools/Export

Beside exporting models from Export Panel you can export models from menu entry **SCS Game Object(s) (.pim)**. It features the same behaviour as export panel with exception that all files will be exported in selected directory.

> **Note:** Even if you can select different directory it still has to be inside current SCS Project Base Path otherwise it would be impossible to reconstruct relative texture paths.

## Export Options

First option here covers scope of export, where you can choose between all three types of export scope similar as in Export Panel#Export modes (Scope).

**Export Scale** - see the chapter Export Panel#Other export options

**Apply Modifiers** - see the chapter Export Panel#Other export options

**Apply Only 'Edge Split' Modifier** - see the chapter Export Panel#Other export options

**Output Format** - see the chapter Export Panel#Other export options

# SCS Tools menu

SCS Blender Tools comprise multiple operators, panels and menus. The **SCS Tools** menu in Blender's topbar provides quick access and search capabilities for actions and properties related to SCS Objects and global configuration. This menu offers an alternative workflow particularly beneficial for newcomers and users preferring centralized access.

The menu is organized into four primary categories:

## Add and misc operators

This section contains menus for adding new SCS Objects, typically accessed via ⇧ Shift + A, alongside miscellaneous actions usually found in the "Operator Search.." menu.

## Sidebar tools

Tools designed for 3D view usage, normally accessible through the 3D view sidebar documented in the SCS Tools Sidebar reference.

## Properties

Popover panels from various Blender properties window tabs, providing access to all SCS Blender Tools settings ranging from Global Settings to object-specific configurations including Materials settings.

## Output management

Popover panels from Blender's properties window Output tab, enabling users to manage and execute export or conversion operations.

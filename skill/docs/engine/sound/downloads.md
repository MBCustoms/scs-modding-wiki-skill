## Sound examples

These sound examples demonstrate techniques to create specific sounds, loops and parameter usage. **Never use these examples directly to create sound bank for your mod!**

Good technique is:
1) create valid your sound bank project from the template
2) copy events from the example project to your project (by FMOD Studio)

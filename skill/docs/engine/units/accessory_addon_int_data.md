# Documentation/Engine/Units/accessory addon int data

## Overview

The **accessory_addon_int_data** unit class is utilized for vehicle accessories that incorporate a visible component with physics-based movement simulation. This class is primarily employed for customizing player vehicles, particularly for dynamic elements such as hanging ornaments or decorative bobbleheads.

## Attributes

Many common attributes are inherited from **accessory_addon_data**.

| Name | Type | Default Value | Description |
|------|------|---------------|-------------|
| data | array<owner_ptr> | <empty> | "Array of the vehicle physics toy data." |

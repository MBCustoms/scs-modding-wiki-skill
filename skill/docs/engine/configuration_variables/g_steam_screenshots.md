# g_steam_screenshots

## Name

`g_steam_screenshots`

## Purpose

"Way in which we should handle the screenshots in steam version."

## Config

global

## Value

integer - handle_way

## handle_way

| Value | Description |
|-------|-------------|
| 0 | no special handling. |
| 1 | screenshots taken by game are also added to steam |
| 2 | game entirely handles capturing of Steams screenshots |

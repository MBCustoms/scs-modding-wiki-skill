# g_rain_reflection

**Name:** g_rain_reflection

**Purpose:** "Rain reflection quality."

**Config:** global

**Value:** integer - reflection_quality

## reflection_quality values

| Value | Description |
|-------|-------------|
| 0 | no reflection |
| 1 | cube reflection |
| 2 | planar with fadeout |
| 3 | planar with cube crossfade |

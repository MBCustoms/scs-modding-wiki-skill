# g_grass_density

**Name:** g_grass_density

**Purpose:** "Grass density setting."

**Config:** global

**Value:** integer - grass_density

## grass_density values

| Value | Description |
|-------|-------------|
| 0 | off |
| 1 | low |
| 2 | medium (high in UI) |
| 3 | high (not available in UI, do not use with 32-bit builds) |

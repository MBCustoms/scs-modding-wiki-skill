# g_hud_speed_limit

**Name:**

g_hud_speed_limit

**Purpose:**

Speed limit state on the adviser.

**Config:**

profile

**Value:**

integer - speed_limit

## speed_limit values

| Value | Description |
|-------|-------------|
| 0 | hide |
| 1 | show truck speed limits |
| 2 | show car/general speed limit |

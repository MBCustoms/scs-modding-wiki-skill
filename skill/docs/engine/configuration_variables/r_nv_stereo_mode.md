# r_nv_stereo_mode

**Name:**

r_nv_stereo_mode

**Purpose:**

"How to handle NV 3D vision stereo mode."

**Config:**

global

**Value:**

integer - stereo_mode_handle_method

## stereo_mode_handle_method values

| Value | Description |
|-------|-------------|
| 0 | disabled |
| 1 | enabled |
| 2 | fully enabled |

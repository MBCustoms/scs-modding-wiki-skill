# g_force_load_selector

**Name:**

g_force_load_selector

**Purpose:**

Forces the appearance of the load slot selector.

**Config:**

profile

**Value:**

integer - load_selector

## load_selector values

| Value | Description |
|-------|-------------|
| -1 | automatic enable as part of crash detection |
| 0 | disabled |
| 1 | enabled |

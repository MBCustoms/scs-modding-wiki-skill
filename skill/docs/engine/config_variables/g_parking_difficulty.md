# g_parking_difficulty

**Name:** g_parking_difficulty

**Purpose:** "Parking difficulty setting."

**Config:** profile

**Value:** integer - difficulty

Correct values are:

| Value | Description |
|-------|-------------|
| 0 | None (showing the selection dialog when reaching destination company) |
| 1 | Easy |
| 2 | Medium or hard |
| 3 | Random |

# r_deferred_debug

**Name:** r_deferred_debug

**Purpose:** "Set deferred renderer debug mode."

**Config:** global

**Value:** integer - debug_mode

## debug_mode values

| Value | Name |
|-------|------|
| 0 | Debug disabled |
| 1 | Position |
| 2 | Normals |
| 3 | Diffuse |
| 4 | Specular |
| 5 | Shadow strength |
| 6 | Specularity power |
| 7 | Additional ambient |
| 8 | Environment map weight |
| 9 | Mix status |
| 10 | Lighting diffuse |
| 11 | Lighting specular |
| 12 | Lighting environment map strength |
| 13 | Composition HDR |
| 14 | Composition LDR |
| 15 | Distortions |
| 16 | MLAA weight |
| 17 | Sunshaft mask |
